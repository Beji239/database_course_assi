# Database Course — T-SQL Assignments

A collection of T-SQL assignments from a relational database course, covering
core SQL concepts applied to a structured data model.

## Topics covered

- **SELECT queries** — filtering, joins, aggregations across multiple tables
- **Stored procedures** — parameterized procedures for data retrieval
- **Encapsulation** — wrapping logic inside procedures and views
- **Conditional logic in SQL** — CASE expressions, IF/ELSE control flow
- **Data modeling** — working with a normalized schema (animal/habitat domain)

## Files

| File | Description |
|------|-------------|
| `2ND_ASSI.sql` – `7TH_ASSI.sql` | Progressive assignments building on each other |
| `LOGIC_IN_SQL.sql` | Conditional logic and control flow |
| `encapsulation.sql` | Stored procedure design and encapsulation patterns |
| `SELECT_FROM_TB_HABITAT_PROC.sql` | Procedure for querying habitat table |
| `GETANIMAL.SQL` | Animal data retrieval queries |
| `SQLQuery1/4/5.sql` | Ad-hoc queries and exploratory work |
